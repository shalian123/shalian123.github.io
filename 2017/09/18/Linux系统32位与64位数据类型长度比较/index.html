Linux系统32位与64位数据类型长度比较

发表于 2017-09-18   |   分类于 linux   |  

Linux系统32位与64位GCC编译器基本数据类型长度对照表
GCC 32位
sizeof(char)=1
sizeof(double)=8
sizeof(float)=4
sizeof(int)=4
sizeof(short)=2
sizeof(long)=4
sizeof(long long)=8
sizeof(long double)=12
sizeof(complex long double)=24

GCC 64位
sizeof(char)=1
sizeof(double)=8
sizeof(float)=4
sizeof(int)=4
sizeof(short)=2
sizeof(long)=8
sizeof(long long)=8
sizeof(long double)=16
sizeof(complex long double)=32

32位，指针占用4个字节；
64位，指针占用8个字节。

size of long max 9223372036854775807
